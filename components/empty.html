# Empty 空状态

当内容为空时，使用空状态组件提示。

# 基本用法

适用大部分默认场景

  <DEmpty />
Expand Copy Copy

# 配置属性

根据场景自定义提示

  <DEmpty text="这里是提示" description="这里是详细说明"/>
Expand Copy Copy
